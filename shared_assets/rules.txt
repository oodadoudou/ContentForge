# --- 规则文件模板 ---
# 格式: 原文 -> 替换内容 (Mode: 模式)
# 模式可以是 Text (普通文本) 或 Regex (正则表达式)。

# --- 在此添加您的规则 ---

# 示例 1: 普通文本替换
旧的名字 -> 新的名字 (Mode: Text)

# 示例 2: 正则表达式，将 "第一章", "第二部" 等替换为带'#'号的形式
# [一二三四五六七八九十百千万]+ 用来匹配中文数字
第([一二三四五六七八九十百千万]+)章 -> #第\1章 (Mode: Regex)
第([一二三四五六七八九十百千万]+)部 -> #第\1部 (Mode: Regex)

# 示例 3: 正则表达式，删除所有行尾的空格
\s+$ -> (Mode: Regex)

# 示例 4: 正则表达式，统一段落开头的缩进（添加两个全角空格）
^([^\s\n]) -> 　　\1 (Mode: Regex)
